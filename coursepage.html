Lab 2: Mobile Application Development Tool

Development Tool
Tool Name: Xamarin
Platform: Xamarin allows you to make applications for multiple platforms. These platforms are iOS, Android, Windows, Mac and more.
Application Development Type: Type:
Xamarin is a Cross-Platform development tool.
Advantages:
The advantages that are gained from being a cross-platform tool is that code share allows you to build an application and use the same code for multiple platforms.
Disadvantages:
The general disadvantages to cross-platform development have been alleviated, meaning that the only real disadvantage is any hardware requirements as needing a mac computer to make iOS applications if using Windows.
Website: https://www.xamarin.com/
Developers: Xamarin has over 350+ employees but not all are developers. Nat Friedman, Miguel de Icaza and Joseph Hill are the founder and co-founders respectively they developed and still are working with Xamarin.
Release Date: The date that I could find closest to the release date was when it was announced on May 16, 2011.
License: The license differs based upon what version of Microsoft Visual Studio you are using along with what classification of whether you are an individual versus a organization. To find the terms that apply to you, they can be found at https://www.xamarin.com/license.
Price: Xamarin is free to download. The only thing you may possibly pay for is a different version of Microsoft Visual Studio (other than Visual Studio Community) which is not associated with the cost of Xamarin.
Programming Language(s): C# and F# for mobile development, where C# is the most common
Features: It has Native like perfomance and is able to access the native api so that native user interfaces can be created. It can share code between types of devices (iOS, Android, Windows Phone). You can use Portable Class Libraries. Using Xamarin and publishing you app you can make you app available to 2.6 billion devices.
Debugger: Uses the Visual Studio Debugger.
Built-in Emulator: Android SDK Emulator
Google Android Manager
Machine Requirements: Either macOS or Windows and Visual Studio for the respective OS. If you have a Windows Machine and are trying to develop a iOS application then you must have a macOS computer on the network (or VM). No real minimum requirements although some features are only available if running certain OS'. Running Windows 10 gives access to all features and macOS Sierra 10.12 for mac.
Integrated Development Enviroment: Xamarin uses Microsoft Visual Studio as it's IDE.
Software Required: The only software required is Visual Studio which will be installed with xamarin if it is not already installed.
Advantages: One IDE and shared code to design applications in multiple platforms. It is free. Simplified maintence since it is shared code. Close to Native Performance.
Disadvantages: If you want the features from the other versions of Microsoft Visual Studio you will need to buy them. Larger App Sizes due to the packaging that Xamarin applies to the application. Delayed Support for latest platform updates.
Language Availability: "English, French, Spanish, German, Chinese, Japanese, Russian, and Brazilian Portuguese languages." (Xamarin, 2018) are the basic languages and more can be added.
Uploading and Selling an App: There are many steps to uploading and selling an app, first of which is making sure that the required documentation is provided the app must describe its functionality and that it has a unique app key. Then once this has been done multiple locations is can be published to, such as Google Play, Amazon App Store, to a website to download, and through email. Each method has specific advantages but their methods should be checked when publishing to make sure that all rules are followed.
Third-Party Emulator: Genymotion is an Android emulator and costs $136/year or more depending on license but appears to be a more common third party emulator that is used. The emulator has some unique features like the ability to emulate the internal drive of a device.
Market Share: I was not able to find a market share number for xamarin.
Number of Apps Available: The actual number of is unknown but for apps in the stores in 2015 it was put at 1.75 million apps with an expected 3.6 million apps at the start of 2018.
Number of Downloads/Users: The number is unknown but was stated for apps that teens download ~6.3 apps per month and adults download 1 app per month which is for all apps regardless of what is was developed in.
Learning: They have a thing called Xamarin University to help teach users to use Xamarin where you can get lectures and other material to learn from. They have two packages for this and 1 is free the other is $83.25USD /month. The paid package gives you the ability to get a certification and 1 on 1 meetings with instructors to get help.
Interesting facts Xamarin says that over 15,000 companies relie on Xamarin such as 3M, HoneyWell, Outback SteakHouse just to name a few.
